Clase 01: Crear y Publicar un Proyecto en GitHub

Resumen y Materiales

Paso 1: Crear una Cuenta en GitHub

1. Visita github.com.

2. Haz clic en "Sign up" y sigue las instrucciones para registrarte.

3. Verifica tu correo electrónico para completar el registro.

Hint: Recomiendo el prompt "Explícame cómo registrarme en GitHub."

Paso 2: Descargar e Instalar GitHub Desktop

1. Visita GitHub Desktop y descarga la aplicación.

2. Sigue las instrucciones de instalación para instalar GitHub Desktop en tu computadora.

3. Inicia sesión en GitHub Desktop con tu cuenta de GitHub.

Hint: Recomiendo el prompt "¿Cómo instalo y configuro GitHub Desktop?"

Paso 3: Crear un Nuevo Repositorio en GitHub

1. Inicia sesión en tu cuenta de GitHub.

2. Haz clic en el ícono "+" en la esquina superior derecha y selecciona "New repository".

3. Llena los detalles del repositorio (nombre, descripción, visibilidad) y haz clic en "Create repository".

Hint: Recomiendo el prompt "¿Cómo creo un nuevo repositorio en GitHub?"

Paso 4: Clonar el Repositorio usando GitHub Desktop

1. Abre GitHub Desktop.

2. En GitHub Desktop, haz clic en "File" y luego en "Clone repository".

3. Selecciona el repositorio que creaste en GitHub y clónalo en tu máquina local.

Hint: Recomiendo el prompt "¿Cómo clono un repositorio usando GitHub Desktop?"

Paso 5: Añadir Archivos y Realizar Commits

1. Añade tus archivos al repositorio clonado en tu computadora.

2. Abre GitHub Desktop y verás los archivos que agregaste.

3. Escribe un mensaje de commit descriptivo en GitHub Desktop y haz clic en "Commit to main".

Hint: Recomiendo el prompt "¿Cómo hago un commit en GitHub Desktop?"

Paso 6: Subir los Cambios a GitHub

1. En GitHub Desktop, haz clic en "Push origin" para subir los cambios a GitHub.

Hint: Recomiendo el prompt "¿Cómo hago un push en GitHub Desktop?"

Paso 7: Habilitar GitHub Pages

1. Ve a la configuración de tu repositorio en GitHub.

2. Desplázate hacia abajo hasta la sección "Pages".

3. En "Source", selecciona la rama que contiene tu proyecto y la carpeta (generalmente "root" o "/docs").

4. Haz clic en "Save" y espera a que se genere tu sitio.

Hint: Recomiendo el prompt "¿Cómo configuro GitHub Pages para mi repositorio?"

Paso 8: Acceder a tu Sitio Web Publicado

1. Una vez que GitHub Pages haya procesado tu proyecto, recibirás la URL de tu sitio web en la misma sección de "Pages" en la configuración del repositorio.

Hint: Recomiendo el prompt "¿Cómo puedo ver mi sitio web en GitHub Pages?"
